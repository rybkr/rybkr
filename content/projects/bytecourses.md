## Overview

Byte Courses is a micro learning management system designed for The Byte Course Project. It provides a simple, focused platform for delivering structured programming courses without the bloat of traditional LMS software.

## Features

- **Course organization**: Structured lessons with progress tracking
- **Markdown content**: Write course material in plain markdown
- **Minimal dependencies**: Single binary deployment
- **Fast and lightweight**: Built for performance, not feature creep

## Technical Details

The backend is written in Go, prioritizing simplicity and maintainability. The frontend uses vanilla HTML and CSS—no JavaScript frameworks required for basic functionality.
